## Installation

1. Clone the repository:
   ```bash
   git clone <URL>
   ```
2. Navigate to the project directory:
   ```bash
   cd project-name
   ```
3. Install the necessary dependencies:

   ```bash
   npm install
   ```

   ## Usage

   To start the project, run the following command:

```bash
npm run dev
```
